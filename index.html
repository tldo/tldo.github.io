为什么要使用占位符呢？
为什么不直接用print(a,b, a + b)

print(a,"+",b, "=",a+b)
# 如果不使用占位符语法，实现相同的效果，是这样的
运算符 描述
[] [:] 下标，切片
** 指数
~ + - 按位取反，正负号
* / % // 乘，除，模，整除
+ - 加，减
>> << 右移，左移
& 按位与
^ | 按位异或，按位或
<= < > >= 小于等于，小于，大于，大于等于
== !== 等于，不等于
is is not 身份运算符
in not in 成员运算符
not or and 逻辑运算符
= += -= = /= %= //= *= &=

%.2f : 这个占位符表示小数点后保留2位数的浮点数
